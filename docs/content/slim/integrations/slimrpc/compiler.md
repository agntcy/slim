# SLIMRPC Compiler

The SLIMRPC compiler generates client and service stub code from Protocol
Buffers definitions for use over SLIM. It works similarly to `protoc`, but
targets SLIM as the transport layer.

For installation, usage, and examples, see the
[SLIMRPC compiler documentation](https://github.com/agntcy/slim-bindings/blob/main/slimrpc-compiler/README.md)
in the [slim-bindings](https://github.com/agntcy/slim-bindings) repository.

For an overview of SLIMRPC, see [SLIMRPC](./index.md).
